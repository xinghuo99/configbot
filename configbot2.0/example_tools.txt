================================================================
  自定义内置 Tool 添加指南
================================================================

一、整体流程
──────────────────────────────────────────────────────────────
  1. 创建 Tool 类（继承 BaseTool）
  2. 在 builtin/__init__.py 中导出
  3. 在 main.py 中注册到 Agent


二、BaseTool 核心要素
──────────────────────────────────────────────────────────────
  每个 Tool 需要实现以下 4 项：

  | 要素 | 说明 |
  |------|------|
  | name | 唯一名称，LLM 和用户通过它调用工具 |
  | description | 功能描述，LLM 用它判断何时调用 |
  | _get_parameters_schema() | 参数 JSON Schema，LLM 用它提取参数 |
  | execute(**kwargs) | 核心逻辑，必须返回 ToolExecResult |


三、示例 1：加密工具（MD5 / SHA256）
──────────────────────────────────────────────────────────────
  文件路径: configbot/tools/builtin/code/encrypt.py

  ```python
  """加密工具"""

  import hashlib
  from typing import Any, Dict

  from ...base import BaseTool, ToolExecResult


  class MD5Tool(BaseTool):
      name = "md5_hash"
      description = "计算文本的 MD5 哈希值"

      def _get_parameters_schema(self) -> Dict[str, Any]:
          return {
              "type": "object",
              "properties": {
                  "text": {"type": "string", "description": "要计算哈希的文本"},
              },
              "required": ["text"],
          }

      async def execute(self, text: str) -> ToolExecResult:
          try:
              result = hashlib.md5(text.encode("utf-8")).hexdigest()
              return ToolExecResult(success=True, data=result)
          except Exception as e:
              return ToolExecResult(success=False, error=str(e))


  class SHA256Tool(BaseTool):
      name = "sha256_hash"
      description = "计算文本的 SHA256 哈希值"

      def _get_parameters_schema(self) -> Dict[str, Any]:
          return {
              "type": "object",
              "properties": {
                  "text": {"type": "string", "description": "要计算哈希的文本"},
              },
              "required": ["text"],
          }

      async def execute(self, text: str) -> ToolExecResult:
          try:
              result = hashlib.sha256(text.encode("utf-8")).hexdigest()
              return ToolExecResult(success=True, data=result)
          except Exception as e:
              return ToolExecResult(success=False, error=str(e))
  ```


四、示例 2：时间日期工具
──────────────────────────────────────────────────────────────
  文件路径: configbot/tools/builtin/system/datetime.py

  ```python
  """时间日期工具"""

  from datetime import datetime, timezone, timedelta
  from typing import Any, Dict

  from ...base import BaseTool, ToolExecResult


  class NowTool(BaseTool):
      name = "get_current_time"
      description = "获取当前日期时间，支持指定时区偏移"

      def _get_parameters_schema(self) -> Dict[str, Any]:
          return {
              "type": "object",
              "properties": {
                  "timezone_offset": {
                      "type": "integer",
                      "description": "时区偏移小时数，默认 8（北京时间）",
                      "default": 8,
                  },
              },
          }

      async def execute(self, timezone_offset: int = 8) -> ToolExecResult:
          try:
              tz = timezone(timedelta(hours=timezone_offset))
              now = datetime.now(tz)
              return ToolExecResult(
                  success=True,
                  data=now.strftime("%Y-%m-%d %H:%M:%S %Z"),
                  metadata={"timezone": f"UTC+{timezone_offset}"},
              )
          except Exception as e:
              return ToolExecResult(success=False, error=str(e))
  ```


五、示例 3：随机数生成工具
──────────────────────────────────────────────────────────────
  文件路径: configbot/tools/builtin/code/random.py

  ```python
  """随机数工具"""

  import random
  import string
  from typing import Any, Dict

  from ...base import BaseTool, ToolExecResult


  class RandomStringTool(BaseTool):
      name = "random_string"
      description = "生成指定长度的随机字符串"

      def _get_parameters_schema(self) -> Dict[str, Any]:
          return {
              "type": "object",
              "properties": {
                  "length": {
                      "type": "integer",
                      "description": "字符串长度，默认 16",
                      "default": 16,
                  },
                  "charset": {
                      "type": "string",
                      "description": "字符集: letters(字母), digits(数字), alphanumeric(字母数字), all(全部)",
                      "default": "alphanumeric",
                  },
              },
              "required": [],
          }

      async def execute(self, length: int = 16, charset: str = "alphanumeric") -> ToolExecResult:
          charsets = {
              "letters": string.ascii_letters,
              "digits": string.digits,
              "alphanumeric": string.ascii_letters + string.digits,
              "all": string.ascii_letters + string.digits + string.punctuation,
          }
          chars = charsets.get(charset, string.ascii_letters + string.digits)
          result = ''.join(random.choice(chars) for _ in range(length))
          return ToolExecResult(success=True, data=result)
  ```


六、注册到 Agent
──────────────────────────────────────────────────────────────
  方式 1（推荐）：在 main.py 的 chat_main() 中注册

  from configbot.tools.builtin.code.encrypt import MD5Tool, SHA256Tool
  from configbot.tools.builtin.system.datetime import NowTool

  agent = Agent()
  agent.register_tool(MD5Tool())
  agent.register_tool(SHA256Tool())
  agent.register_tool(NowTool())


  方式 2：在 builtin/__init__.py 中导出

  # configbot/tools/builtin/__init__.py
  from .code.encrypt import MD5Tool, SHA256Tool
  from .system.datetime import NowTool

  __all__ = [
      ...,
      "MD5Tool", "SHA256Tool", "NowTool",
  ]

  然后在 Agent._setup_builtin_tools() 中统一注册（批量方式）。


七、效果验证
──────────────────────────────────────────────────────────────
  注册后，_print_welcome() 会自动展示新工具。

  无 LLM 模式（关键词匹配）：
    > 帮我计算 "hello" 的 md5
    > 生成一个随机密码

  LLM 模式（AI 推理）：
    > 对 "admin123" 做 sha256 加密
    > 现在北京时间是几点？

  两种模式都会自动路由到对应工具。


八、ToolExecResult 说明
──────────────────────────────────────────────────────────────
  @dataclass
  class ToolExecResult:
      success: bool            # 是否执行成功
      data: Any = None         # 返回数据
      error: Optional[str] = None  # 错误信息
      metadata: Dict = {}      # 附加元数据（不影响输出）
